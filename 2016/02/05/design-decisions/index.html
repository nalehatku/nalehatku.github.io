Design decisions

2016-02-05

Project structure

UI - generated by express framework
BL - inspired by MongoDb course (university.mongodb.com)

Promise API for BL

  • Q preferred over CujoJS due to 10x more usage (downloads from npm)
  • when (cujo) does not have scope on their nodefn.apply/call, ‘q’ does (Q.ninvoke). (see https://github.com/cujojs/when/issues/263)

Hash handling

  • “#” means new report
  • “#/code” means opening report code

history overiding

“#” -> “#” NO (navigation using api, not changing #)
“#” -> “#/a” NO
“#/a” -> “#/b” YES
“#/a” -> “#/a” NO (navigation using api, not changing #)
“#/a” -> “#” YES

save report YES
